<br>A "complete" website template how a website can be structured.

## About the project
This repo is supposed to work as a templete and "starter-kit" for websites. The structure gives dynamic capabilites such as changing, removing and adding paths.

## Architecture Overview
This project is built into two main layers:

### /frontend:
At the moment only:
<br> 
* "/frontend/static" 
<br>exists. Here are the .css files placed.
### /backend:
* /templates
   <br> Here is the base.html that gives the website its ground structure.
* /routes
    <br> This is were the python programes are stored that routes the user to the different path in you website.
* /programs
    <br> If you want any programs (ex: python), this is were they are stored.
